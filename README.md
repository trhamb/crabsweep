# crabsweep
A system for scraping and parsing questions from Tom Scott's 'Lateral' Podcast.

As a huge fan of the [Lateral Podcast](https://lateralcast.com), I'm often relaying questions I've heard to friends. The problem is that I have to spend so long reading the transcripts on the website in order to find the questions and their respective answers, so I figured I'd make a system to help future me be lazy :)
